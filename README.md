# Pixel Roguelike (Godot 4)

A fast-paced 2D pixel-art roguelike built in Godot.

## Features
- Wave-based enemy system
- Mid-weight assault weapon mechanics
- Procedural enemy spawning
- Smooth 2D camera system

## Tech Stack
- Godot 4
- GDScript
- C++ (planned for performance modules)
- Python (asset pipeline tools)
